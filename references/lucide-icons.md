# Lucide Icon Snippets

Copy-paste-ready inline SVG for the icons you'll reach for most often. All snippets share these attributes:

```
xmlns="http://www.w3.org/2000/svg"
viewBox="0 0 24 24"
fill="none"
stroke="currentColor"
stroke-width="2"
stroke-linecap="round"
stroke-linejoin="round"
```

The starter template's `.card-icon` and `.icon-button` classes size them to 20px and 18px respectively. Inline `width`/`height` only if you need an override.

If you need an icon that isn't here, look it up at https://lucide.dev/icons/ and copy the SVG. Don't load the CDN script — paste the markup.
